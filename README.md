### 4. 코드블럭
```
def main():
    price = 1000  # 상품 가격
    goods = 3  # 구매 상품 수
    total = 0  # 총 가격
    for i in range(1, goods + 1):  # 구매 상품 수가 3인 경우 1~3 반복 수행
        print("2+1 상품입니다.")
        if i % 3 == 0:  # 3의 배수인 경우 가격을 더하지 않음
            continue
        total += price
    print("총 가격은 " + str(total) + "원입니다.")
main()
```
